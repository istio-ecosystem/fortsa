# Contributing to Fortsa

As this is a young project, all forms of contribution are welcome. Contributors must
adhere to all community codes of conduct. Contributions to code should try their best
to conform to the guidelines laid out in AGENTS.md.

Contributions come in the form of, but not limited to, bug reports, pull requests,
discussions, recommendations, and updates to code, documentation, and other artifacts.

If you have any questions, feel free to contact the maintainers of this project. Currently
that is Steve Scaffidi, who can be reached via email stephen+fortsa@scaffidi.net, or via
the project's GitHub issues and discussions.
